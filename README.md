# Задание 2 — API-пайплайн: данные → LLM → результат

## Описание задачи

Написан скрипт на языке Go, который автоматически обрабатывает входные данные и отправляет их в языковую модель через API.

В качестве задачи выбрано извлечение характеристик товаров из текстовых описаний. На вход программе подается CSV-файл `products.csv`, где каждая строка содержит номер товара и его текстовое описание. Далее программа отправляет эти данные в модель Nemotron 3 через OpenRouter API. Модель извлекает из текста бренд, категорию, цену и валюту товара. Полученный ответ сохраняется в файл `result.json`.

## Инструкция запуска

Установить Go

```bash
go version
```

Создать файл .env в проекте:

```env
OPENROUTER_API_KEY=Ваш_ключ
```

Проверить файл products.csv с колонками:

```csv
id,description
```

Запустить скрипт:

```bash
go run .
```

Результат сохранится в файл:

```text
result.json
```

## Пример входных данных

Файл `products.csv` содержит 30 строк с описаниями товаров.

Пример строк из файла:

```csv
id,description
1,"Apple iPhone 15 128GB black smartphone, price 799 USD"
2,"Samsung Galaxy A55 256GB blue Android phone, cost 420 USD"
3,"Nike Air Max 90 men's sneakers white, price 135 USD"
```

## Пример выходных данных

После запуска создается файл `result.json`. 

Пример одного элемента результата:

```json
{
  "id": 1,
  "brand": "Apple",
  "category": "smartphone",
  "price": 799,
  "currency": "USD"
}
```
